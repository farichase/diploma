# REFAL SMT-SCPstr


### Запуск
```sh
	git clone https://github.com/farichase/diploma.git
	cd SuperCompiler/my-app
	chmod +x setup.sh
	./setup.sh
	npm install
	sudo docker build -t "my_image" .
```	
##### Запуск сервера (консоль №1):
```sh
	cd SuperCompiler/my-app/backend
	sudo node app
```	
##### Запуск клиента (консоль №2):
```sh
	cd SuperCompiler/my-app
	npm start
```
